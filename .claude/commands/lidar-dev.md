# LidarApp Development Commands

## /lidar-dev:new-service <name>
Creates a new service with protocol, implementation, and tests.

## /lidar-dev:new-view <name>
Creates a new SwiftUI view with ViewModel.

## /lidar-dev:phase-status
Shows current development phase progress.

## /lidar-dev:run-tests
Runs all tests for the project.
